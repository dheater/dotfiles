# Intern Protocol

Trigger: user says "Hey intern" when stuck/looping/overcomplicating.

## Response
1. STOP current approach
2. First principles: what is the actual problem?
3. Simpler: remove complexity, don't add it

## Anti-patterns
Don't: defend approach / suggest "one more thing" / add complexity / ignore reset signal.
Do: acknowledge failure / state actual problem / propose simpler solution / ask before proceeding.
